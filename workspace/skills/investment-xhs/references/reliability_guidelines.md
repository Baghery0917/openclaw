# 信息可靠性评估指南

## 可靠性等级定义

### A级：最高可靠性
**定义**：来自官方、权威、可验证的第一手资料。

**包含来源**：
1. 公司官方文件
   - 财务报表（SEC filings、年报、季报）
   - 官方新闻稿
   - 投资者关系页面公告
   - 公司官网技术文档

2. 政府监管机构
   - 证监会（SEC）备案文件
   - 专利局授权专利
   - 工商注册信息（天眼查、企查查官方数据）

3. 权威数据库
   - Crunchbase Pro数据
   - PitchBook Verified数据
   - Bloomberg终端数据

**验证要求**：
- 必须提供直接链接或文件引用
- 时间戳必须在1年以内（财务报表可接受最近季度）
- 如为PDF等文件，需注明页码

### B级：高可靠性
**定义**：来自权威第三方机构的分析报道。

**包含来源**：
1. 专业财经媒体
   - 华尔街日报（WSJ）
   - 金融时报（FT）
   - 彭博社（Bloomberg）
   - 路透社（Reuters）

2. 科技行业媒体
   - TechCrunch
   - The Information
   - 36氪（深度报道）
   - 钛媒体
   - 晚点LatePost

3. 分析师报告
   - 摩根士丹利、高盛等投行报告
   - IDC、Gartner行业分析
   - 知名咨询公司报告（麦肯锡、BCG）

4. 学术论文
   - 同行评审期刊
   - 顶级会议论文集

**验证要求**：
- 作者需为署名记者或分析师
- 媒体需有良好声誉记录
- 报道需包含具体数据引用

### C级：中等可靠性
**定义**：来自一般媒体报道或第三方数据平台。

**包含来源**：
1. 一般科技媒体
   - 腾讯科技、新浪科技
   - 网易科技、搜狐科技
   - 创业邦、创业家

2. 社交媒体官方账号
   - 公司官方微博
   - 创始人认证Twitter/LinkedIn
   - 官方微信公众号

3. 数据平台
   - 天眼查、企查查（非官方验证数据）
   - IT桔子、烯牛数据
   - 拉勾、BOSS直聘招聘数据

**验证要求**：
- 需交叉验证（至少2个独立来源）
- 注意发布时间和时效性
- 警惕营销软文性质内容

### D级：低可靠性
**定义**：未经证实或匿名来源信息。

**包含来源**：
1. 匿名爆料
   - 脉脉匿名区
   - 知乎匿名回答
   - 微博匿名爆料

2. 未经核实传闻
   - 微信群聊截图
   - 未验证的邮件截图
   - 小道消息

3. 营销号内容
   - 标题党自媒体
   - 无来源转载
   - AI生成内容

**使用限制**：
- 仅可作为参考线索，不能作为分析依据
- 必须明确标注可靠性为D级
- 不能用于核心论点支撑

## 交叉验证方法

### 三级验证体系
1. **一级验证**：同一信息在2个A级来源中一致
   - 可靠性自动提升至A+
   - 可作为核心论据

2. **二级验证**：A级+B级来源一致
   - 可靠性为A级
   - 可作为主要论据

3. **三级验证**：B级+C级来源一致
   - 可靠性为B级
   - 需谨慎使用，注明验证级别

### 矛盾信息处理
当不同来源信息矛盾时：

1. **优先顺序**：A级 > B级 > C级 > D级
2. **时效性**：优先采用最新信息
3. **来源权威**：优先采用更权威来源
4. **利益相关**：注意来源可能存在的利益冲突

## 时效性要求

### 不同信息类型的有效期
1. **财务数据**：
   - 季度财报：有效期为下一季度财报发布前
   - 年度财报：有效期为下一年度财报发布前

2. **用户增长数据**：
   - 有效期为3个月
   - 需注明数据统计截止日期

3. **技术信息**：
   - 专利信息：长期有效
   - 产品发布：有效期为1年
   - 技术路线图：有效期为6个月

4. **团队信息**：
   - 核心人员任职：有效期为6个月
   - 团队规模：有效期为3个月

### 过时信息处理
1. **超过有效期但无更新**：
   - 标注“最后更新日期”
   - 注明“此信息可能已过时”
   - 不作为主要决策依据

2. **历史数据趋势分析**：
   - 可使用历史数据，但需注明时间范围
   - 对比分析时确保时间区间一致

## 特殊情况的处理

### 未公开公司的信息
对于未公开披露财务的初创公司：

1. **估算方法**：
   - 根据融资额和股权比例估算估值
   - 根据团队规模估算成本
   - 根据产品定价和用户数估算收入

2. **可靠性标注**：
   - 所有估算必须标注“基于公开信息估算”
   - 注明估算方法和假设条件
   - 可靠性等级不超过C级

### 中英文信息差异
1. **优先采用来源**：
   - 公司注册地官方语言版本优先
   - 如为中美两地公司，优先采用SEC filings英文版

2. **翻译准确性验证**：
   - 关键数据需核对原文
   - 专业术语需确认准确翻译

## 评估记录格式

### 信息条目格式
```
【{可靠性等级}】{信息内容}（来源：{具体来源}，时间：{YYYY-MM-DD}）
```

### 示例
```
【A级】2023年Q4营收1.2亿美元，同比增长45%（来源：SEC 10-K filing，2024-02-15）
【B级】据TechCrunch报道，公司MAU已达500万（来源：TechCrunch文章，2024-01-20）
【C级】天眼查显示公司拥有12项发明专利（来源：天眼查专利页面，2024-03-10）
```
