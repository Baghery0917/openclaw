# Investment XHS - 小红书投资分析报告内容策划

## 概述

本技能实现完全自主的小红书投资分析报告内容策划流水线。每天自动选择一家AI科技公司,从五个维度(技术、增长、成本、竞争、团队)收集信息,评估可靠性,生成深度投资分析报告,并转化为小红书平台适用的标题、图片、摘要和标签。

## 工作流概览

完整的SOP包含9个步骤,每一步都需要人工审核以确保质量:

1. **公司选择与审核**:从维护的AI科技公司名单中选取一家,通过微信发送给用户审核
2. **五维信息收集**:从技术、增长、成本、竞争、团队五个角度搜集具体依据
3. **信息可靠性评估**:对搜集的信息进行可信度评级
4. **资料审核**:将搜集的信息、来源和可信度发送给用户二次审核
5. **深度投资分析**:调用DeepSeek Reasoner模式生成投资分析报告
6. **小红书标题生成**:基于报告生成5个专业投研圈视角的爆款标题
7. **HTML报告渲染**:将报告用专业投资分析风格排版并渲染为图片
8. **文案摘要生成**:生成易读、带emoji的小红书文案摘要
9. **标签生成与交付**:生成5个相关标签,交付标题、图片、摘要、标签

### 步骤1:公司选择与审核

**目标**:从AI科技公司名单中选择一家尚未分析或超过1年未分析的公司。

5. 将选中的公司通过微信发送给用户审核:

### 步骤2:五维信息收集

**目标**:从五个维度收集具体、可验证的信息。

**五个维度及收集要点**:

1. **技术**:
   - 自研能力:查看公司官网、技术博客、GitHub仓库
   - 数据壁垒:数据来源、数据规模、数据独特性
   - 专利数:查询专利数据库(中国专利网、USPTO等)
   - 是否为"套壳"产品:检查核心算法是否原创

2. **增长**:
   - ARR(年度经常性收入):查找财务报表、投资者演示稿
   - 用户增长:月度活跃用户(MAU)、季度增长率
   - 获客成本(CAC):市场营销支出/新增客户数
   - 留存率:用户留存数据(如果有公开数据)

3. **成本**:
   - 推理成本占收入比重:估算计算资源消耗
   - 算力储备:GPU数量、云服务合同
   - 毛利率:收入-成本结构分析

4. **竞争**:
   - 差异化优势:独特功能、专利技术、商业模式
   - 客户切换成本:用户迁移难度
   - 巨头覆盖风险:是否容易被大公司功能更新替代

5. **团队**:
   - 核心人员背景:创始人、CTO的学术和职业经历
   - 工程落地经验:团队是否具备产品化能力
   - 人才结构:科学家 vs. 工程师 vs. 销售比例

**收集工具**:
- **Tavily API**（默认）：使用AI优化的专业搜索API，提供高质量、结构化的搜索结果。支持技术、财务、竞争情报等多维度信息收集。
- `web_search`：备选搜索引擎，当Tavily不可用时使用
- `web_fetch`：抓取官网、技术文档
- `browser`：登录专业数据库（如Crunchbase、PitchBook）

3. 支持搜索模式：`tavily`（默认）、`web_search`、`mock`（模拟数据）

### 步骤3:信息可靠性评估

**目标**:对收集的每一条信息进行可信度评级。

**可靠性等级**:
- **A级(最高)**:官方财务报表、SEC filings、公司官网公告
- **B级(高)**:权威媒体报道(36氪、钛媒体、TechCrunch)、行业分析报告
- **C级(中)**:第三方数据平台(天眼查、企查查)、社交媒体官方账号
- **D级(低)**:匿名爆料、未经证实的传闻

**评估方法**:
1. 交叉验证:多个独立来源确认同一信息
2. 时效性:优先使用最近1年内的信息
3. 来源权威性:优先选择知名媒体、分析师报告

**关键文件**:
- `scripts/reliability_evaluator.py`:自动评估脚本
- `references/reliability_guidelines.md`:详细评估指南

### 步骤4:资料审核

**目标**:将收集的信息、来源和可信度整理发送给用户审核。

**输出格式**:
```markdown

...

...
```

**交付方式**:通过微信发送完整信息汇总,等待用户确认。

### 步骤5:深度投资分析

**目标**:调用DeepSeek Reasoner模式进行深入分析。

**操作流程**:
1. 将审核通过的信息整理为上下文
2. 使用 `sessions_spawn` 创建Reasoner会话:
   ```python
   sessions_spawn(
       runtime='acp',
       agentId='deepseek-reasoner',
       task=f'基于以下信息生成AI公司投资分析报告:{context}',
       model='deepseek/deepseek-reasoner',
       thinking='on'
   )
   ```
3. 生成报告结构:
   - 公司概况
   - 五维深度分析
   - 风险提示
   - 投资建议(买入/持有/卖出)
   - 目标估值区间

**关键文件**:
- `scripts/analysis_generator.py`:报告生成脚本
- `references/report_template.md`:报告结构模板

### 步骤6:小红书标题生成

**目标**:生成5个专业投研圈视角的爆款标题。

**Prompt模板**:
```
请基于这份投资分析报告,为我在小红书(专业投研圈视角)生成5个具有高度专业感、能够吸引投资经理和合伙人点击的标题。
要求:
1. 突出核心投资亮点
2. 使用专业术语但不过于晦涩
3. 有吸引力的数字和对比
4. 包含情绪价值词汇
5. 适合小红书平台风格
```

**交付方式**:通过微信发送5个标题供用户选择。

**关键文件**:
- `scripts/title_generator.py`:标题生成脚本
- `references/xhs_title_prompts.md`:标题示例库

### 步骤7:HTML报告渲染

**目标**:将投资分析报告用专业风格排版并渲染为图片。

**操作流程**:
1. 使用HTML/CSS模板(`assets/report_template.html`)
2. 将报告内容填充到模板中
3. 使用浏览器工具渲染并截图:
   ```python
   browser(action='open', url='file:///path/to/report.html')
   browser(action='screenshot', fullPage=True, type='png')
   ```
4. 保存图片文件

**关键文件**:
- `scripts/html_renderer.py`:HTML渲染脚本
- `assets/report_template.html`:报告HTML模板
- `assets/style.css`:专业投资分析样式

### 步骤8:文案摘要生成

**目标**:生成易读、带emoji的小红书文案摘要。

**要求**:
- 控制在500字以内
- 使用适当的emoji增加可读性
- 突出核心投资观点
- 适合小红书用户阅读习惯

**示例格式**:
```

📊 技术实力:自研架构,专利护城河
💰 增长迅猛:ARR年增300%
⚠️ 风险提示:巨头入场竞争
...
```

**关键文件**:
- `scripts/summary_tag_generator.py`:摘要生成脚本

### 步骤9:标签生成与交付

**目标**:生成5个相关标签并交付最终成果。

**标签要求**:
- 涵盖公司领域、技术方向、投资阶段
- 适合小红书话题标签
- 有搜索热度

**最终交付物**:
1. 选定的标题
2. 渲染后的报告图片
3. 文案摘要
4. 5个相关标签

**交付方式**:通过微信一次性发送所有成果。

## 资源文件说明

本技能包含以下资源文件,用于支持完整的自动化流程:

### scripts/ 脚本目录

包含执行具体任务的Python脚本:

1. `company_manager.py` - 公司名单管理
   - 功能:读取/更新公司名单,检查重复,添加新公司
   - 数据结构:JSON格式,包含公司名称、分析日期、融资阶段等

2. `research_collector.py` - 五维信息收集
   - 功能:自动化搜索五个维度的信息,整理为结构化数据
   - 依赖:web_search, web_fetch, browser工具

3. `reliability_evaluator.py` - 信息可靠性评估
   - 功能:根据来源类型自动评级,支持交叉验证

4. `analysis_generator.py` - 深度投资分析生成
   - 功能:调用DeepSeek Reasoner模式,生成专业投资报告

5. `title_generator.py` - 小红书标题生成
   - 功能:基于报告内容生成5个爆款标题
   - 模板:使用预定义的prompt模板

6. `html_renderer.py` - HTML报告渲染
   - 功能:将报告填充到HTML模板,使用浏览器渲染截图

7. `summary_tag_generator.py` - 摘要和标签生成
   - 功能:生成小红书文案摘要和5个相关标签
